# Russian's Delight

**Russian's Delight** is an addon for [Farmer's Delight Refabricated](https://modrinth.com/mod/farmers-delight-refabricated) that adds authentic Russian cuisine to Minecraft. Dive into the world of hearty soups, golden pancakes, fresh cucumbers, and shareable feasts!

## About

This is the first release of the mod. It may receive updates with new content, fixes, and improvements in the future.

Found a bug or have a suggestion? Report it [here](https://github.com/AmongFox/Russian-s-Delight/issues).

## Dependencies

- [Farmer's Delight Refabricated](https://modrinth.com/mod/farmers-delight-refabricated) (>= 1.20.1-2.1.0)
- [Fabric API](https://modrinth.com/mod/fabric-api)
